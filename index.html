Tenho experiência com desenvolvimento web utilizando HTML, CSS e JavaScript, além de sólida base em lógica de programação com Python e domínio em banco de dados MySQL. Estou sempre buscando aplicar o que aprendo em projetos práticos, com o objetivo de evoluir como desenvolvedor e contribuir com ideias que façam a diferença. Na seção “Sobre mim”, compartilho com mais profundidade minha formação e os cursos que venho realizando. Já concluí meus estudos em lógica de programação, o que me proporcionou uma base sólida para compreender algoritmos e resolver problemas de forma estruturada. Logo abaixo, você verá os ícones das tecnologias que fazem parte do meu repertório técnico — HTML, CSS, JavaScript, Python e SQL — representando as ferramentas que venho dominando e aplicando em meus estudos e projetos.
